Blue buttons on the right correspond to the different pokedex modes.

Modes are:
- Mode Select
- Credits
- Settings (metric system or not)
- Browse mode (by number)
- Who's that pokemon?

- Battle simulator
- Browse mode (alphabetical)
- Browse mode (by weight)
- Browse mode (by height)
- Party

- Evolutions
- Basic Info
- Weight/Height
- Locations on Map
